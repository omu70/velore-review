How to use these creatives

1. Asset folder: All 7 product photos and 9 hotel logos are real, pulled from historylifestyle.in's Shopify CDN, saved alongside this file in historylifestyle_assets/. Move that folder with this HTML and everything renders.
2. Export to PNG/JPG: Open this file, screenshot each .canvas at 1080×1080. Or right-click → "Save Page As" → Complete to keep image links intact.
3. For Meta Ads Manager: upload the canvas screenshot as the creative; use the "Primary text" copy at the top of each ad and the headline + description from headline-block.
4. Dynamic version: Ad 03 (multi-product grid) and Ad 05 (cart card) should be served as Meta DPA from your Shopify catalog so the actual product the user viewed renders automatically.
5. Brand: Off-white #F4F1EC background, #0F0F0F ink, champagne accent #B89968 — pulled from the live site's hospitality-luxury palette.
